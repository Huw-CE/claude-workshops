Skills · 20 min

Teach Claude
how we work.

Skills are structured markdown files that give Claude specialist knowledge for a specific task. Install one and Claude can run full UX research cycles, generate design systems, write developer handoff specs, or compress your token usage by 65%. They can trigger automatically when Claude detects a relevant task, or you can invoke them manually with a slash command.

Live Activity · 10 min

Build your own skill.

We're building a "Studio Brief Intake" skill together — one that asks all the right questions before any new CE project kicks off. Client name, sector, audience, deliverables, timeline, tone. You'll leave with a skill that works every time you start a new brief.

Steps

How to create a skill in Claude.

1. Open Claude Desktop → click your name → Customize

2. Navigate to the Skills tab

3. Click Create Skill

4. Give it a name: Studio Brief Intake

5. Paste the skill definition we're writing together

6. Save — Claude will now run this skill automatically when you start a new project brief

"A brilliant skill you forget to invoke is worse than a mediocre one that auto-triggers. Design for invocation, not just capability."
