Before we get into prompting technique, let's get oriented. The desktop app has more going on than it looks. Spend five minutes here and you'll stop missing half the features.

The Side Panel

Your home base.

Panel features What it does How to use it
Chats Every conversation lives here. Claude keeps your history, but don't treat it like memory — each new chat starts fresh unless you're in a Project. Use chats for one-off tasks. Name them if they matter. Start a new chat and name it "Infosys Expo Brief." Find it again in the sidebar. This is your filing system — use it.
Projects Projects are Claude's version of a persistent working relationship. Everything inside a Project shares context — uploaded files, instructions, prior conversations. If you're working on an ongoing client, a campaign, or any multi-session task, it belongs in a Project. Think of it as a dedicated briefing room that Claude never forgets. Create a Project called "Charles Elena Studio." Set a system prompt: "You are working with Charles Elena, a Melbourne digital experience agency."
Artifacts When Claude produces something substantial — a document, a table, a piece of code, a presentation — it drops it into an Artifact. Artifacts live in a side panel, can be iterated on independently, and exported. If Claude produces something worth keeping, it'll likely end up here automatically. Ask Claude to "put this into an Artifact" on your next output. Then try editing it directly in the artifact panel — without re-prompting.
Customize This is where you teach Claude who you are — your role, your preferences, your tone. We'll go deep on this in the Skills session, but for now: open it, read what's there, and add one line about what you do at Charles Elena. Add: "I'm a designer at Charles Elena in Melbourne. I work on digital experience, brand, and UX." Watch the next response shift.

Settings Worth Knowing

Under the hood.

Settings overview What it does Worth doing
Usage Shows your message consumption across the billing period. Useful for understanding which model you're burning through fastest, and when to switch down to Sonnet or Haiku to conserve your limit. Check it after today's session to see how many messages the workshop consumed across models. You'll start to feel where the limits are.
Connectors This is where MCP tools live — Figma, Google Drive, Gmail, Calendar, Zapier. We'll connect these properly in the MCP session, but take a look now so you know where to come back to. Find Figma in the list now. Don't connect yet — but note the path: Settings → Connectors. We'll activate it together later.
Import memory Switching from ChatGPT? Claude can import your ChatGPT memory directly — the preferences, context, and habits you've built up carry over. Worth doing before your first real session if you've been living in ChatGPT. Do this tonight if you've been using ChatGPT. Your context transfers. Starting Claude cold when you had a trained ChatGPT is leaving value on the table.

Chat Functions

More than a text box.

Chat tools What it does Try this
Research Invoke Research mode when you need Claude to go broad before going deep — gathering sources, synthesising information, building a picture before giving you an answer. It takes longer but the output is significantly more considered. Use it for competitive analysis, trend reports, or any brief where you'd normally spend an hour on Google first. Ask: "Research the top AI-powered consumer experience trends at live events in 2025." Compare the depth, structure, and source quality to a standard response.
Use Style Claude can adapt its writing to match a specific voice. Choose from built-in styles or upload a sample of your own writing and Claude will mirror it. Useful for client copy that needs to sound like a specific person, or for maintaining a consistent tone across a campaign. In the Skills session we'll go further — locking tone, compression, and format into reusable skills. Paste a paragraph of your own writing and ask: "Match this voice and write a short intro paragraph for an AI experience at a tech expo."
