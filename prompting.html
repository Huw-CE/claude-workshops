The Side Panel

Your home base.

Panel features
Chats
Every conversation lives here. Claude keeps your history, but don't treat it like memory — each new chat starts fresh unless you're in a Project. Use chats for one-off tasks. Name them if they matter.
Projects
Projects are Claude's version of a persistent working relationship. Everything inside a Project shares context — uploaded files, instructions, prior conversations. If you're working on an ongoing client, a campaign, or any multi-session task, it belongs in a Project. Think of it as a dedicated briefing room that Claude never forgets.
Artifacts
When Claude produces something substantial — a document, a table, a piece of code, a presentation — it drops it into an Artifact. Artifacts live in a side panel, can be iterated on independently, and exported. If Claude produces something worth keeping, it'll likely end up here automatically. You can also ask Claude to put anything into an Artifact explicitly.
Customize
This is where you teach Claude who you are — your role, your preferences, your tone. We'll go deep on this in the Skills session, but for now: open it, read what's there, and add one line about what you do at Charles Elena. It'll immediately improve every response you get.

Settings Worth Knowing

Under the hood.

Settings overview
Usage
Shows your message consumption across the billing period. Useful for understanding which model you're burning through fastest, and when to switch down to Sonnet or Haiku to conserve your limit.
Connectors
This is where MCP tools live — Figma, Google Drive, Gmail, Calendar, Zapier. We'll connect these properly in the MCP session, but take a look now so you know where to come back to.
Import memory from other AI providers
Switching from ChatGPT? Claude can import your ChatGPT memory directly — the preferences, context, and habits you've built up carry over. Worth doing before your first real session if you've been living in ChatGPT.

Chat Functions

More than a text box.

Chat tools
Research
Invoke Research mode when you need Claude to go broad before going deep — gathering sources, synthesising information, building a picture before giving you an answer. It takes longer but the output is significantly more considered. Use it for competitive analysis, trend reports, or any brief where you'd normally spend an hour on Google first.
Use Style
Claude can adapt its writing to match a specific voice. You can choose from built-in styles or — and this is the interesting part — you can upload a sample of your own writing and Claude will mirror it. Useful for client copy that needs to sound like a specific person, or for keeping a consistent tone across a campaign.

Styles are a good start. In the Skills session, we'll show you how to go further — locking tone, compression level, and output format into reusable skills like Caveman that trigger automatically.
